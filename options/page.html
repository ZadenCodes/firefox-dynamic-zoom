Dynamic Zoom Options

Dyanamic Zoom will adjust the zoom level so that the webpage thinks it's window is this wide. This should not necessarily be the width of your monitor, unless you love whitespace. Default: 1280 pixels

Dynamic Zoom will not zoom a website more than this amount. Lower values will give larger windows more room to breath, but may cause text re-flow on some websites. Default: 300%

Toggle Dynamic Zoom
Ctrl+Shift+Left
Decrease Simulated Width
Ctrl+Shift+Down
Increase Simulated Width
Ctrl+Shift+Up
